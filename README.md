# Books Library App
A **React + Redux Toolkit Query** application for managing a library of books. Supports creating, editing, deleting, and searching books.

---

## Tech Stack

- **React 18** – Frontend library
- **Redux Toolkit Query (RTK Query)** – Data fetching & cache management
- **React Router v6** – Routing
- **TypeScript** – Static typing
- **CSS** – Dark theme styling
- **Vite / CRA** – Project bootstrapping (if using Vite/CRA)
- **REST API** – Backend: [Books Backend](https://books-backend-teal.vercel.app/api/)
